ðŸ“Œ Setup Instructions:
-----------------------
1. Import database schema from `database.sql` file.
2. Update database credentials in `application/config/database.php`.
3. Configure base_url in `application/config/config.php`.
4. Enable hooks in `application/config/config.php`:
      $config['enable_hooks'] = TRUE;
5. Add hook in `application/config/hooks.php`:
      $hook['post_controller_constructor'][] = array(
          'class'    => 'Auth_hook',
          'function' => 'check_access',
          'filename' => 'Auth_hook.php',
          'filepath' => 'hooks'
      );
6. Run project in browser:
      http://localhost/ci3_rbac/
